# Alex GUI

> 一个真正"好看 + 好用 + 快速"的本地媒体库桌面应用

------------------------------------------------------------------------

## 🧠 为什么做这个？

大多数本地媒体管理工具都有这些问题：

-   ❌ UI 老旧、信息密度混乱
-   ❌ 操作路径复杂，上手成本高
-   ❌ 性能一般，浏览卡顿

**Alex GUI 的目标很简单：**

> 做一个"你愿意每天打开"的媒体库工具

------------------------------------------------------------------------

## ✨ 核心体验

### 🎬 沉浸式媒体浏览

-   卡片式布局，视觉干净
-   封面优先，降低认知负担
-   浏览体验接近流媒体平台

------------------------------------------------------------------------

### 🖼️ 极致预览体验

-   点击即放大
-   支持键盘操作（← → ESC）
-   非循环切换，符合直觉
-   单图自动隐藏切换按钮

👉 细节不是功能，是体验

------------------------------------------------------------------------

### ⚡ 本地原生性能

-   无网络依赖
-   启动快、响应快
-   资源占用可控

------------------------------------------------------------------------

### 🎨 现代 UI 设计

-   左侧导航 + 右侧内容区
-   卡片式信息结构
-   视觉层级清晰
-   持续优化中（不是套模板）

------------------------------------------------------------------------

## 🧱 技术架构

不是 Electron 的"网页壳"，而是更轻量的方案：

### 前端

-   React
-   TypeScript

### 桌面端

-   Go
-   Wails（轻量桌面框架）

👉 优势： - 更小体积 - 更低内存占用 - 更接近原生应用体验

------------------------------------------------------------------------

## 🚀 功能一览

-   📁 本地媒体库管理

-   🔍 快速浏览内容

## 📦 快速开始

### 1. 克隆项目

``` bash
git clone https://github.com/StarEllis/Alex-gui.git
cd Alex-gui
```

### 2. 安装依赖

``` bash
cd frontend
npm install
```

确保已安装： - Go - Wails CLI

------------------------------------------------------------------------

### 3. 启动开发环境

``` bash
wails dev
```

------------------------------------------------------------------------

### 4. 打包应用

``` bash
wails build
```

生成桌面应用（Windows / Mac / Linux）。

------------------------------------------------------------------------
